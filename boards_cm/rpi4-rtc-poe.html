RPi4-RTC-PoE DIN rail mini server

Website More Info
More Info GitHub Issue

Description

The RPi4-RTC-PoE DIN rail mini server offers PoE power, Gigabit Ethernet, USB 2.0, USB-over-Ethernet, an M.2 2230 slot for NVMe SSDs, a power button, status LEDs, an RTC, hardware watchdog, RS485, and ZigBee, and is a very compact little DIN server.
